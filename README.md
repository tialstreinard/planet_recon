# Planet Recon
Automated PI planet recon for Eve Online Planetary Industry.

## Notes
- The script provided is tailored to Storm Planets
- Modify the script to suit your needs (eg different planet type, starting system, distance, planet threshold)
- You can find the required System ID in many ways, one is to search for the System in ZKillboard and copy-pasting the ID you find in the URL

## How to run
`./find_storms.sh <PLANETID> <NUMBER OF JUMPS>`
